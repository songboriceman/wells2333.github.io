centos6系统环境搭建，包括安装jdk、mysql、tomcat等。

安装jdk

检查是否已经安装jdk：rpm -qa | grep java

卸载系统自带jdk：rpm -qa | grep java | xargs rpm -e –nodeps

下载jdk：

wget --no-cookie --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u151-b12/e758a0de34e24606bca991d704f6dcbf/jdk-8u151-linux-x64.rpm?AuthParam=1515326179_35a6fe79c9e9d731cead18775e21ee01 -O jdk-8u151-linux-x64.rpm

安装jdK：rpm -ivh jdk-8u151-linux-x64.rpm

检查：rpm -qa | grep java

java -version
